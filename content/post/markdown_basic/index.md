## 1. 标题 headline/title/

使用 # ，可以表示 1-6 级标题

# 一级标题 (# 一级标题)

## 二级标题 (## 二级标题)

### 三级标题 (### 三级标题)

#### 四级标题 (#### 四级标题)

##### 五级标题 (##### 五级标题)

###### 六级标题 (###### 六级标题)

## 2. 区块引用 blockquote/quote/

> 引用内容 （>引用内容）

## 3. 代码区块 code block/code/

\``` c 指定以下代码为 C 语言

void main()

{

printf("Hello, Markdown!");

}

\```

效果：

```c
void main()
{
printf("Hello, Markdown!");
}
```
